# MyApp Web Dashboard

This project build using Stack:

- Vite
- Vue3
- TailwindCSS
- Vee validate
- Yup
- Vue toastification
- Lucide vue next (Icon)
- Vue router
- Axios
- Platzi Fake Store API

## Scripts

- How to Run `=>` `npm run dev`
- How to Build `=>` `npm run build`
- How to Lint `=>` `npm run lint`

Link `=>` https://dashboard-vuenite-web.vercel.app
